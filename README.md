## ✅ 実装済み機能

### 🖊️ ウェブ編集モード（`?edit` パラメータ）
URLの末尾に `?edit` を追加するとどのページでも編集モードが有効になります。

| URL例 | 説明 |
|---|---|
| `index.html?edit` | ホームページを編集 |
| `route.html?edit` | 行程ページを編集 |
| `company_information.html?edit` | 会社情報ページを編集 |

**機能詳細：**
- 画面上部に編集ツールバーが表示される
- テキストをクリックすると直接インライン編集可能（黄色の枠で強調）
- **「保存」ボタン**：編集内容を LocalStorage に保存（次回アクセス時も反映）
- **「リセット」ボタン**：そのページの全変更を初期化
- **「終了」ボタン**：編集モードを終了し通常表示に戻る
- 保存した内容は `?edit` なしの通常URLでも自動反映される
- 実装ファイル：`js/edit-mode.js`
